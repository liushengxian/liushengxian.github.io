# AI Robot 讨论会

一个有趣的AI对话模拟器，可以创建三个不同角色的AI机器人，围绕一个话题展开10轮深度讨论。

## 功能特点

1. **创建三个AI角色**：为每个机器人设置名称、角色设定和发言风格
2. **深度讨论**：三个AI轮流发言，进行10轮讨论
3. **智能回复**：支持两种回复生成模式：
   - **DeepSeek API模式**：使用真实的AI生成回复
   - **模拟模式**：使用内置的智能模拟回复
4. **自动总结**：讨论结束后自动生成总结报告
5. **本地存储**：所有设置保存在本地浏览器中

## 使用步骤

### 1. 设置AI角色
- 为每个机器人设置：
  - **名称**：如"智慧学者"、"创新先锋"、"实践专家"
  - **角色设定**：描述机器人的特点、专业领域和思维方式
  - **发言风格**：选择正式、友好、批判性、创造性或幽默风格

### 2. 配置API（可选）
- 如果需要使用真实的AI回复：
  1. 访问 [DeepSeek平台](https://platform.deepseek.com/api_keys) 获取API Token
  2. 在设置面板中启用"使用DeepSeek API生成回复"
  3. 输入API Token并点击"测试API连接"
  4. 连接成功后，讨论将使用DeepSeek API生成回复

### 3. 开始讨论
1. 输入讨论话题（如："人工智能对教育的影响"）
2. 点击"开始讨论"按钮
3. 观看三个AI机器人轮流发言
4. 可以随时暂停或重置讨论

### 4. 查看总结
- 10轮讨论结束后，系统会自动生成：
  - 讨论要点总结
  - 各机器人发言次数统计
  - 总讨论轮数和发言次数

## 技术特点

### 回复生成机制
- **API模式**：将角色设定和讨论上下文发送到DeepSeek API，生成符合角色特点的回复
- **模拟模式**：基于预设的回复模板和角色特点，智能生成多样化的回复

### 字数限制
- 每个AI的发言严格控制在140字以内
- 确保讨论内容简洁、聚焦

### 数据安全
- API Token仅存储在本地浏览器中
- 不发送任何用户数据到外部服务器（除API调用外）

## 常见问题

### Q: 没有API Token可以使用吗？
A: 可以！应用内置了智能模拟回复系统，无需API Token也能体验完整功能。

### Q: API调用会收费吗？
A: DeepSeek API有免费额度，超出后可能会产生费用。请参考DeepSeek官方定价。

### Q: 讨论可以中途暂停吗？
A: 可以，点击"暂停"按钮可以暂停讨论，点击"继续"可以恢复。

### Q: 数据会保存吗？
A: 所有角色设置和API Token都保存在本地浏览器中，刷新页面不会丢失。
