screen常用的初级命令

Command Effects
screen -S [screen-name] 新建指定名称的screen并进入
screen -list 列出现存的screen
screen -r [screen-name] 连接到指定名称的screen
screen -X -S [screen-name] quit 退出指定名称的screen

screen内常用的命令

Ctrl+A d 退出当前的screen
